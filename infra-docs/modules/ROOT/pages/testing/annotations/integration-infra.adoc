[[integration-testing-annotations-spring]]
= Infra Testing Annotations

The TODAY Framework provides the following set of Infra-specific annotations that you
can use in your unit and integration tests in conjunction with the TestContext framework.
See the corresponding javadoc for further information, including default attribute
values, attribute aliases, and other details.

Infra testing annotations include the following:

* xref:testing/annotations/integration-infra/annotation-bootstrapwith.adoc[`@BootstrapWith`]
* xref:testing/annotations/integration-infra/annotation-contextconfiguration.adoc[`@ContextConfiguration`]
* xref:testing/annotations/integration-infra/annotation-webappconfiguration.adoc[`@WebAppConfiguration`]
* xref:testing/annotations/integration-infra/annotation-contexthierarchy.adoc[`@ContextHierarchy`]
* xref:testing/annotations/integration-infra/annotation-contextcustomizerfactories.adoc[`@ContextCustomizerFactories`]
* xref:testing/annotations/integration-infra/annotation-activeprofiles.adoc[`@ActiveProfiles`]
* xref:testing/annotations/integration-infra/annotation-testpropertysource.adoc[`@TestPropertySource`]
* xref:testing/annotations/integration-infra/annotation-dynamicpropertysource.adoc[`@DynamicPropertySource`]
* xref:testing/annotations/integration-infra/annotation-dirtiescontext.adoc[`@DirtiesContext`]
* xref:testing/annotations/integration-infra/annotation-testexecutionlisteners.adoc[`@TestExecutionListeners`]
* xref:testing/annotations/integration-infra/annotation-recordapplicationevents.adoc[`@RecordApplicationEvents`]
* xref:testing/annotations/integration-infra/annotation-commit.adoc[`@Commit`]
* xref:testing/annotations/integration-infra/annotation-rollback.adoc[`@Rollback`]
* xref:testing/annotations/integration-infra/annotation-beforetransaction.adoc[`@BeforeTransaction`]
* xref:testing/annotations/integration-infra/annotation-aftertransaction.adoc[`@AfterTransaction`]
* xref:testing/annotations/integration-infra/annotation-sql.adoc[`@Sql`]
* xref:testing/annotations/integration-infra/annotation-sqlconfig.adoc[`@SqlConfig`]
* xref:testing/annotations/integration-infra/annotation-sqlmergemode.adoc[`@SqlMergeMode`]
* xref:testing/annotations/integration-infra/annotation-sqlgroup.adoc[`@SqlGroup`]
* xref:testing/annotations/integration-infra/annotation-disabledinaotmode.adoc[`@DisabledInAotMode`]
* xref:testing/annotations/integration-infra/annotation-beanoverriding.adoc#infra-testing-annotation-beanoverriding-testbean[`@TestBean`]
* xref:testing/annotations/integration-infra/annotation-beanoverriding.adoc#infra-testing-annotation-beanoverriding-mockitobean[`@MockitoBean` and `@MockitoSpyBean`]
