[[infra-testing-annotation-contextcustomizerfactories]]
= `@ContextCustomizerFactories`

`@ContextCustomizerFactories` 用于为特定测试类、其子类及其嵌套类注册 `ContextCustomizerFactory` 实现。如果你希望全局注册工厂，应通过 xref:testing/testcontext-framework/ctx-management/context-customizers.adoc[`ContextCustomizerFactory` 配置] 中描述的自动发现机制进行注册。

以下示例展示了如何注册两个 `ContextCustomizerFactory` 实现：

[tabs]
======
Java::
+
[source,java,indent=0,subs="verbatim,quotes",role="primary"]
----
@ContextConfiguration
@ContextCustomizerFactories({CustomContextCustomizerFactory.class, AnotherContextCustomizerFactory.class}) // <1>
class CustomContextCustomizerFactoryTests {
  // class body...
}
----
<1> 注册两个 `ContextCustomizerFactory` 实现。

======


默认情况下，`@ContextCustomizerFactories` 支持从超类或封闭类继承工厂。有关示例和更多详细信息，请参见 xref:testing/testcontext-framework/support-classes.adoc#testcontext-junit-jupiter-nested-test-configuration[`@Nested` 测试类配置] 和 {today-framework-api}/test/context/ContextCustomizerFactories.html[`@ContextCustomizerFactories` javadoc]。
