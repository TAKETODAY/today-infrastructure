[[infra-testing-annotation-sql]]
= `@Sql`

`@Sql` 用于注解测试类或测试方法，以配置在集成测试期间针对给定数据库运行的 SQL 脚本。以下示例展示了如何使用它：

[tabs]
======
Java::
+
[source,java,indent=0,subs="verbatim,quotes",role="primary"]
----
@Test
@Sql({"/test-schema.sql", "/test-user-data.sql"}) // <1>
void userTest() {
  // 运行依赖于测试架构和测试数据的代码
}
----
<1> 为此测试运行两个脚本。
======

有关更多详细信息，请参见 xref:testing/testcontext-framework/executing-sql.adoc#testcontext-executing-sql-declaratively[使用 @Sql 声明式执行 SQL 脚本]。
