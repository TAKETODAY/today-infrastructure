[[infra-testing-annotation-beforetransaction]]
= `@BeforeTransaction`

`@BeforeTransaction` 表示对于已配置为使用 Infra `@Transactional` 注解在事务中运行的测试方法，被注解的 `void` 方法应在事务开始前运行。`@BeforeTransaction` 方法不需要是 `public` 的，并且可以在基于 Java 8 的接口默认方法上声明。

以下示例展示了如何使用 `@BeforeTransaction` 注解：

[tabs]
======
Java::
+
[source,java,indent=0,subs="verbatim,quotes",role="primary"]
----
@BeforeTransaction // <1>
void beforeTransaction() {
  // 在事务开始前运行的逻辑
}
----
<1> 在事务之前运行此方法。
