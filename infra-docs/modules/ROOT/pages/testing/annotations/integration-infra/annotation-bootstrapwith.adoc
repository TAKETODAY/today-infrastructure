[[infra-testing-annotation-bootstrapwith]]
= `@BootstrapWith`
:page-section-summary-toc: 1

`@BootstrapWith` 是一个类级别的注解，你可以使用它来配置如何引导 Infra TestContext 框架。具体来说，你可以使用 `@BootstrapWith` 来指定自定义的 `TestContextBootstrapper`。有关更多详细信息，请参见 xref:testing/testcontext-framework/bootstrapping.adoc[引导 TestContext 框架] 部分。
