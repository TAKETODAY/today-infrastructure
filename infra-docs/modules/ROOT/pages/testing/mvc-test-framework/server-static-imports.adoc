[[mvc-test-server-static-imports]]
= 静态导入
:page-section-summary-toc: 1

当直接使用 MockMvc 执行请求时，你需要以下静态导入：

- `MockMvcBuilders.{asterisk}`
- `MockMvcRequestBuilders.{asterisk}`
- `MockMvcResultMatchers.{asterisk}`
- `MockMvcResultHandlers.{asterisk}`

记住这一点的一个简单方法是搜索 `MockMvc*`。如果使用 Eclipse，请务必也将上述内容添加为 Eclipse 首选项中的“`favorite static members`”。

当通过 xref:testing/webtestclient.adoc[WebTestClient] 使用 MockMvc 时，你不需要静态导入。`WebTestClient` 提供了一个流畅的 API，无需静态导入。
