[[mvc-ann-httpentity]]
= HttpEntity

`HttpEntity` 与使用 xref:web/webmvc/mvc-controller/ann-methods/requestbody.adoc[`@RequestBody`]
或多或少相同，但它是基于一个公开请求头和正文的容器对象。以下清单显示了一个示例：

[tabs]
======
Java::
+
[source,java,indent=0,subs="verbatim,quotes",role="primary"]
----
@PostMapping("/accounts")
public void handle(HttpEntity<Account> entity) {
  // ...
}
----
