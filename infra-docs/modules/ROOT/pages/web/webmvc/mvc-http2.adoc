[[mvc-http2]]
= HTTP/2
:page-section-summary-toc: 1

Servlet 4 容器需要支持 HTTP/2，TODAY Framework 5 与 Servlet API 4 兼容。
从编程模型的角度来看，应用程序不需要做任何特定的事情。但是，这涉及到服务器配置方面的考虑。
有关更多详细信息，请参阅 {today-framework-wiki}/HTTP-2-support[HTTP/2 wiki 页面]。

Servlet API 确实暴露了一个与 HTTP/2 相关的构造。您可以使用 `jakarta.mockApi.http.PushBuilder`
主动向客户端推送资源，它支持作为 `@RequestMapping` 方法的
xref:web/webmvc/mvc-controller/ann-methods/arguments.adoc[方法参数]。
