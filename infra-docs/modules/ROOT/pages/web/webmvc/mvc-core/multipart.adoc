[[mvc-multipart]]
= Multipart 解析器

`infra.web.multipart` 包中的 `MultipartParser` 是用于解析 `multipart` 请求（包括文件上传）的策略。
有一个默认的的 `infra.web.multipart.parsing.DefaultMultipartParser` 实现。底层来源于
（Apache Commons FileUpload）


要启用 `multipart` 处理，您需要在 IoC 容器声明一个名为 `MultipartParser` bean。
当接收到内容类型为 `multipart/form-data` 的 POST 时，解析器会解析内容。
