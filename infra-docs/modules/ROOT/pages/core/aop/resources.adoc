[[aop-resources]]
= 更多资源
:page-section-summary-toc: 1

有关 AspectJ 的更多信息，请访问 {aspectj-site}[AspectJ 网站]。

由 Adrian Colyer 等人撰写的 _Eclipse AspectJ_ (Addison-Wesley, 2005) 提供了
AspectJ 语言的全面介绍和参考。

Ramnivas Laddad 撰写的 _AspectJ in Action_, Second Edition (Manning, 2009) 强烈推荐。
该书重点关注 AspectJ，但也深入探讨了许多通用的 AOP 主题。
