[[expressions-array-construction]]
= 数组构造

你可以使用熟悉的 Java 语法来构建数组，可选地提供一个初始化器以便在构造时填充数组。
下例展示了如何做到这一点：

[tabs]
======
Java::
+
[source,java,indent=0,subs="verbatim,quotes",role="primary"]
----
	int[] numbers1 = (int[]) parser.parseExpression("new int[4]").getValue(context);

	// 带初始化器的数组
	int[] numbers2 = (int[]) parser.parseExpression("new int[] {1, 2, 3}").getValue(context);

	// 多维数组
	int[][] numbers3 = (int[][]) parser.parseExpression("new int[4][5]").getValue(context);
----
======

[NOTE]
====
构造多维数组时，目前不能提供初始化器。
====

[CAUTION]
====
任何构造数组的表达式——例如，通过 `new int[4]` 或 `new int[] {1, 2, 3}`——都无法编译。
有关详细信息，请参阅 xref:core/expressions/evaluation.adoc#expressions-compiler-limitations[编译器限制]。
====
