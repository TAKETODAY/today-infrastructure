[[jmx]]
= JMX

Infra 中的 JMX (Java Management Extensions) 支持提供的功能使您可以轻松透明地将 Infra 应用程序集成到 JMX 基础设施中。

.JMX?
****
本章不是 JMX 的介绍。它并不试图解释您为什么要使用 JMX。如果您是 JMX 的新手，请参阅本章末尾的 xref:integration/jmx/resources.adoc[更多资源]。
****

具体来说，Infra JMX 支持提供四个核心功能：

* 任何 Infra Bean 作为 JMX MBean 的自动注册。
* 控制 Bean 管理接口的灵活机制。
* 通过远程 JSR-160 连接器声明性地公开 MBean。
* 本地和远程 MBean 资源的简单代理。

这些功能旨在无需将您的应用程序组件耦合到 Infra 或 JMX 接口和类即可工作。实际上，在大多数情况下，您的应用程序类无需知道 Infra 或 JMX 即可利用 Infra JMX 功能。
